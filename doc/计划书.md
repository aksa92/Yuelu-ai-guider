# 岳麓山AI导游 — 实战学习计划书

## 一、项目总览

### 1.1 这是什么

基于 Spring Boot + DeepSeek API 的"岳麓山AI导游"，支持问答、RAG 知识检索、工具调用、Agent 自主决策、实时位置服务。

### 1.2 技术栈

| 层 | 选型 | 原因 |
|---|---|---|
| 后端框架 | Spring Boot 3.5 + Java 21 | 已搭建 |
| AI 框架 | 手写 → LangChain4j | 先学原理再用框架 |
| 大模型 | DeepSeek API（deepseek-v4-flash） | 已接入 |
| 向量存储 | 手动实现（List + 余弦相似度） | 学习原理，后续可替换 |
| 地图服务 | 高德地图 API | 国内主流 |
| 前端 | 待定（先专注后端） | |
| 数据库 | 待定（有需要再加） | |

### 1.3 学习目标

从"懂 Transformer 原理"到能独立：
- 设计并实现一个完整的 AI 应用
- 掌握 RAG 全链路及优化方法
- 掌握 Agent 架构设计
- 能应对 AI 方向的面试

## 五、一些规则

### 节奏
- 每晚 1 小时左右
- 每次先讨论实现逻辑，确认后再写代码
- 每个 Phase 结束后可以停下来回顾、调整后续计划

### 调整机制
- 某个方向想深入 → 加晚数
- 某个方向觉得够了 → 砍晚数或跳过去
- 遇到面试需要重点准备某个考点 → 提前对应 Phase
